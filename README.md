# DevOps CD Pipeline — Simulated Banking Environment

A practice repository built to mirror a real-world production CD pipeline 
structure used in a South African banking project.

## Context
I was provided the exact file structure and real pipeline challenges 
from the production banking environment. I independently troubleshot and 
resolved these issues, building hands-on experience with real-world 
DevOps problems.

## What I worked on
- Built and debugged CD pipeline scripts in a banking-grade file structure
- Troubleshot real pipeline issues from a production AWS environment
- Worked with AWS CloudFormation templates for infrastructure provisioning
- Wrote and fixed Bash scripts for deployment automation
- Configured AppSpec files for AWS CodeDeploy

## Tech stack
- AWS (CloudFormation, CodeDeploy)
- Bash scripting
- AppSpec / CodeDeploy configuration
- Git / GitHub
